# vizsgajelentkezo
 Used by students at BME to sign up for Databases exam with me.

## Design
- admin
  - list students by date
  - add and remove exam times
  - kick out student from exam
- student
  - sign up for exam
    - email
    - name
    - available date
  - withdraw from exam
  - only allow one exam at a time
- exam time
  - max 4 applicants
  - add or remove
  - set max number of applicants
